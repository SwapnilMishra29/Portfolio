WonderLust

WonderLust is a full-stack CRUD-based marketplace for property listings. Users can create, edit, and delete listings with image uploads and secure authentication. Role-based authorization is implemented with Passport.js and session cookies. Client and server-side validation is done using Bootstrap and Joi respectively. Cloudinary is used for CDN-based image handling, optimizing image load. The application includes a review system with a 5-star widget and emoji comments. CI/CD is integrated via GitHub Actions and deployment is on Render.
